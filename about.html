p5.graphing.js

A library for p5.js
By Ethan DeGuire

About This Project

p5.graphing.js was made by Ethan DeGuire, as a standalone library for p5.js


It consists of one class, "Graph", which is currently capable of creating xy plane graphs, with future plans for pie charts, histograms, box and whisker charts, and more.


The source code for this project can be found on GitHub
